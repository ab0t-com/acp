# Release Downloads

Public release archives for ACP, mirrored here for visibility + support.

Preferred install:

```bash
curl -fsSL https://raw.githubusercontent.com/ab0t-com/acp/main/install.sh | bash
```

Archive naming:

```
latest.txt
acp_<version>_<os>_<arch>.tar.gz   # contains: acp, coordd, acp-mcp, LICENSE, README
checksums.txt
```

The installer prefers GitHub Releases, then falls back to these mirrored files.
